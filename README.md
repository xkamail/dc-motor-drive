# DC Motor Driver Practice

- Controller: ATMega328P

- Driver: TB6552FNG


## Functions:
  select control mode: preset (25%, 50%, 75%, 100%), manual (potentio)
  
  select ccw, cw for m1 and m2 ( toggle switch )
  
  short brake individual ( two emergency button )
  
  short brake all
  
  stop and stanby
